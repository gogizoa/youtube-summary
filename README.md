# YouTube Summary Generator

## 프로젝트 소개
이 프로젝트는 YouTube 동영상을 자동으로 요약해주는 애플리케이션입니다. AI 기술을 활용하여 긴 동영상 콘텐츠를 간결하고 의미 있는 요약으로 변환합니다.

## 주요 기능
- YouTube 동영상 크롬익스텐션을 통한 자동 요약
- 키워드 및 주요 포인트 추출
- 요약 결과 복사기능

## 기술 스택
- 백엔드: Python (FastAPI)
- 프론트엔드: React.js
- AI 모델: OpenAI GPT
- 데이터 처리: YouTube Data API
- 배포: Docker, AWS/GCP

## 시작하기

### 필수 요구사항
